# kenning-web

Landing page for [Kenning](https://github.com/kenninglabs) — hosted on GitHub Pages at **https://kenning.in**.

Static, single-page, zero build step.

## Files
- `index.html` — the whole page (self-contained; only external asset is `icon.svg`).
- `icon.svg` — logo + favicon.
- `CNAME` — binds the GitHub Pages site to `kenning.in`. **Do not delete.**

## Publish
GitHub → repo **Settings → Pages** → Source: deploy from branch `main`, folder `/ (root)`.
The `CNAME` file wires the custom domain automatically.
